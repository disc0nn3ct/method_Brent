# method_Brent
Метод Брента 



Используется библиотека [NTL](https://github.com/libntl/ntl)

Установка: 
```shell

   % gunzip ntl-xxx.tar.gz
   % tar xf ntl-xxx.tar
   % cd ntl-xxx/src
   % ./configure 
   % make
   % make check
   % sudo make install

```

Компиляция и запуск программы:
```shell
make
```

Для удаления: 
```shell
make clean
```
